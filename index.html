Privacy Policy for Flutter Snooper

Last updated: August 4, 2025

1. Introduction

Flutter Snooper ("we," "our," or "us") is committed to protecting your privacy. This Privacy Policy explains how our Android application collects, uses, and protects information when you use our service.

2. Information We Collect

2.1 Device Information

Flutter Snooper analyzes and displays the following information about Flutter applications installed on your device:

  • Flutter SDK versions used by installed applications
  • Package names and metadata of Flutter applications
  • Application installation information visible through standard Android APIs
  • Public application information that does not require special permissions

2.2 No Personal Data Collection

We do NOT collect:

  • Personal identifying information
  • User accounts or login credentials
  • Location data
  • Contact information
  • Files or documents from your device
  • Network traffic or communication data

3. How We Use Information

The application uses the collected information solely to:

  • Display Flutter SDK versions and package information locally on your device
  • Provide analysis and insights about Flutter applications on your device
  • Compare installed Flutter versions with available releases

4. Data Storage and Security

4.1 Local Storage

All analyzed data is stored locally on your device using a local database. No information is transmitted to external servers or third parties.

4.2 Remote Data

The application may download publicly available Flutter SDK release information from official sources (GitHub) to provide version comparisons. This does not involve sending any of your device information.

5. Data Sharing

We do not share, sell, rent, or trade any information collected by Flutter Snooper with third parties. All data remains on your device.

6. Permissions

Flutter Snooper only uses standard Android permissions to:

  • Query installed applications (QUERY_ALL_PACKAGES or package visibility)
  • Access internet for downloading public Flutter release information

We do not request or use any sensitive permissions that would access personal data.

7. Data Retention

All data collected by the application is stored locally on your device. You can clear this data at any time by:

  • Uninstalling the application
  • Clearing the application's data through Android settings

8. Children's Privacy

Flutter Snooper does not collect any personal information and is safe for users of all ages. The application only displays technical information about installed applications.

9. Changes to This Privacy Policy

We may update this Privacy Policy from time to time. We will notify you of any changes by posting the new Privacy Policy on this page and updating the "Last updated" date.

10. Third-Party Services

Flutter Snooper may access publicly available information from:

  • GitHub (flutter/flutter repository) for Flutter release information

These services have their own privacy policies, which we encourage you to review.
